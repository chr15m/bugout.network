The Bugout Box is a decentralized web appliance. It's a server device that you control, reachable from a web browser anywhere in the world.

It's a censorship-resistant way to serve data, APIs and apps.

What can you do with your Bugout Box?

  • Seed WebTorrents.
  • Run decentralized Bugout server apps.
